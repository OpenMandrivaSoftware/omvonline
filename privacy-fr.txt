
Quelles données personnelles collectons-nous ?

Diverses informations sont collectées dans différents endroits de nos sites Web.
Vous trouverez ci-dessous une vue d'ensemble des données que nous conservons.

Premièrement, nous enregistrons votre adresse email, notre nom et votre adresse
 postale.
Deuxièmement, nous vous donnons un numéro de client unique. Ce numéro est
 utilisé pour ajouter des informations liées à votre compte Mandrake comme :
 	- votre souscription au mailing-list
	- lorsque vous enregistrez un produit acheté, nous vous demandons des informations personnelles ainsi que des détails sur votre configuration pour pouvoir délivrer le meilleur support, personnalisé pour vos besoins.
	- sur notre page de donation et sur notre site de vente en ligne (MandrakeStore), nous enregistrons les données pour pouvoir effectuer la facturation (nous ne conservons pas de numéro de carte de crédit).

Que faisons-nous avec ces informations ?

Nous ne partageons jamais vos informations personnelles avec d'autres personnes ou d'autres sociétés.
Nous utilisons ces données stritement pour les raisons suivantes :

	- si vous avez souscrit à une mailing-list, nous conservons cette adresse email dans notre base de données pour vous donner des informations personnalisées et/ou pour vous permettre de participer aux discussions de nos différentes mailing-lists.
	Si vous envoyez un message sur une telle liste, d'autres personnes verront votre adresse email mais aucune autre partie n'a accès à notre base de données d'abonnés. A n'importe quel moment, vous pouvez vous désabonner de n'importe quelle liste en utilisant la méthode appropriée.
	- si vous fournissez des informations lorsque vous enregistrez un produit pour recevoir du support, nous conservons cette information privée sauf s'il est nécessaire à un de nos partenaires pour fournir le service que vous avez demandé.
	- si vous utilisez MandrakeStore, nous utilisons votre numéro de client pour suivre votre achat et délivrer votre produit.
	- si vous entrez des informations marketing, nous les conservons pour un usage interne uniquement pour améliorer nos produits. Ainsi ils pourront être plus proche de vos besoins que vous achetiez un produit ou non.

	Comment changer/corriger vos informations personnelles ?

	La loi française donne aux personnes le droit d'accéder, corriger, modifier ou supprimer les données personnelles que nous détenons.
	Nous offrons cela dans le monde entier. vous pouvez exercer ce droit à n'importe quel moment en vous loggant sur votre compte avec votre numéro de client et en éditant vos données personnelles.
	Vous pouvez vous logguer sur cette page : http://www.mandrakeonline.net
